Ultima Online - Sphereserver 56b plugin to PSPAD
================================================
Plugin, which extends the program PSPad. This is a plugin that adds syntax for ultima online sphereserver 56b, supports over 1,500 orders and to help with the description gives over 1300 command. Suitable for developers old game Ultima Online.
 
1) Installing the plugin
----------------------------------

### Sphereserver.ini     
copied to the folder:

    C:\Program Files\PSPad editor\Syntax\

 
### SphereServer.DEF
copied to the folder:

    C:\Program Files\PSPad editor\Context\


2) Setting color syntax
----------------------------------
set the color syntax is best shown in Figure `setting_color.PNG` , but it's inviduálním settings.
